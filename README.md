<h1 align="center">HouseCleaner - Android Service Management App</h1>

<p align="center">
  <img src="https://github.com/HimanthaD4/House-Cleaner-Android-App/blob/main/HouseCleanerApp/app/src/main/res/drawable/banner.webp" alt="HouseCleaner Banner" width="100%">
</p>

<p align="center">
  <strong>A fully functional Android app for managing house cleaning services.</strong>
</p>

---

<h2>✨ Key Features</h2>
<ul>
  <li>👥 <strong>User Roles</strong>: Separate interfaces for house owners and cleaners.</li>
  <li>📋 <strong>Job Management</strong>: Owners can post cleaning jobs with details like location, rooms, bathrooms, and floor type.</li>
  <li>💰 <strong>Automatic Pricing</strong>: The system calculates cleaning fees based on house details (rooms, bathrooms, and floor type).</li>
  <li>✅ <strong>Job Status Tracking</strong>: Owners can track job statuses (available/accepted) and toggle them easily.</li>
  <li>📷 <strong>Image Upload</strong>: Owners can upload house images for better job descriptions.</li>
  <li>💬 <strong>Comments Section</strong>: Cleaners and owners can communicate through a comments section for each job.</li>
  <li>🗄️ <strong>Database Integration</strong>: Uses SQLite for local data storage with efficient CRUD operations.</li>
</ul>

---

<h2>🛠️ Technologies Used</h2>
<ul>
  <li>☕ <strong>Java</strong>: Core programming language for Android development.</li>
  <li>🗃️ <strong>SQLite</strong>: Local database for storing house details, user information, and comments.</li>
  <li>📜 <strong>RecyclerView</strong>: Efficiently displays lists of houses and jobs.</li>
  <li>🎨 <strong>Material Design</strong>: Clean and intuitive UI components.</li>
  <li>🖼️ <strong>Base64 Encoding</strong>: Handles image uploads and storage.</li>
</ul>
